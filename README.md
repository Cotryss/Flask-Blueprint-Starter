<h1>🚀 Flask Blueprint Starter</h1>

<p>
A clean and scalable <strong>Flask starter template</strong> built with
Blueprint architecture and ready for production projects.
</p>

<h2>✨ Features</h2>

<ul>
    <li>Blueprint architecture</li>
    <li>Application Factory pattern</li>
    <li>SQLAlchemy integration</li>
    <li>Flask-Login authentication</li>
    <li>Responsive Login & Register pages</li>
    <li>Flash message component</li>
    <li>Database auto initialization</li>
    <li>Clean and organized project structure</li>
</ul>

<h2>📂 Project Structure</h2>

<pre>
app/
├── blueprints/
├── utils/
├── startup/
├── templates/
├── static/
├── app.py
├── models.py
├── extensions.py
└── config.py
</pre>

<h2>📦 Ready To Use</h2>

<p>
Clone the repository, install the requirements, configure your
<code>.env</code> file and start building your application.
</p>

<h2>🛠 Tech Stack</h2>

<p>
Flask • SQLAlchemy • Flask-Login • Tailwind CSS
</p>
